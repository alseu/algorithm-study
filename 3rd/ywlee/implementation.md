# 구현(Implementation)

'머릿속에 있는 알고리즘을 소스코드로 바꾸는 과정'

'풀이를 떠올리는 것은 쉽지만 소스코드로 옮기기 어려운 문제'를 구현 유형의 문제라고 한다.

'완전탐색' : 모든 경우의 수를 다 계산

'시뮬레이션' : 문제에서 제시한 알고리즘을 한 단계씩 차례대로 직접 수행

## 고려해야할 사항들

- 자료형의 크기
- 리스트의 크기
  - 메모리 제한
    | int 자료형 데이터의 개수(리스트의 길이) | 메모리 사용량 |
    | --------------------------------------- | ------------- |
    | 1,000                                   | 약 4KB        |
    | 1,000,000                               | 약 4MB        |
    | 10,000,000                              | 약 40MB       |
